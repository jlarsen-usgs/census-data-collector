Introduction
============

The Census Data Collector is a Python package that queries TigerWeb and the U.S. Census
API to download data. Users are able to provide geospatial boundaries to the
Census Data Collector code and pull and resample data that conforms to their study
area(s). Currently the Census Data Collector supports pulling data from the following
U.S. Census products
   - American Community Survey 1 year and 5 year data tables
   - Decennial Summary File 3 data

Additional data products may be supported in the future, and communit contributions
are welcome to the project..

Return to the Github `Census Data Collector <https://github.com/jlarsen-usgs/census-data-collector>`_ website.

Installation
------------

To install the development version of the Census data collector from the git repository type:

.. code-block:: bash

    pip install git+https://github.com/jlarsen-usgs/census-data-collector.git
